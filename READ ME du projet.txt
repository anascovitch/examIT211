
Design Agency 

Ce projet est une application web développé avec Django permet de montrer et de partager les travaux d'une association de photographe en vue de sa visibilité.
 Ils mettent leurs photos en ligne et créent des posts à leur sujet sur un blog Ce site permet d'avoir deux types d’utilisateurs,
les abonnés et les créateurs. Seuls les créateurs doivent pouvoir créer du
contenu. Ce contenu est partagé dans un flux social, et les abonnés
peuvent choisir quels créateurs ils veulent suivre.

Démarrage Rapide

Pour démarrer ce projet en local, vous devez disposer des outils suivants ou similaires:
-xampp permet démarrer le serveur apache et mysql en mode local afin d'avoir accès à la base de données. 
Le fichier de base données est joint au dossier et il est nommé:"monprojet_bd
Il a été exporté donc assuré vous de l'avoir importé dans votre php my admin premièrement

Voici le nom d'utilisateur et mot de passe pour avoir accès à la base de données :
utilisateur: root
mot de passe:


-Un IDE, comme vscode : ouvrer le dossier monprojet et lancer le serveur django:
python manage.py runserver
Assurez vous que vous soyez dans le bon dossier sinon faites la cd monprojet
